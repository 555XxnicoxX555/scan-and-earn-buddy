# Decisiones - Proyecto Gastronomia Sumi

## 2026-06-07

- El canal `#proyecto-gastronomia` se toma como espacio de trabajo del proyecto.
- El canal fue movido a la categoria Discord `Proyectos`.
- El proyecto se organiza internamente bajo `projects/gastronomia-sumi/`.
- La idea principal sera `Sumi Cafe`: negocio gastronomico ficticio con fidelizacion por puntos.
- El sistema Sumi se presenta como diferencial comercial para aumentar recompra, frecuencia y ticket promedio.
